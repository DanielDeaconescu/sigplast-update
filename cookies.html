Politică de confidențialitate

Această politică de confidențialitate se aplică tuturor utilizatorilor paginii de internet https://www.sigplast.ro.

Ce este un cookie?

Un "internet cookie" (numit și "browser cookie", "HTTP cookie" sau pur si simplu "cookie") este un fișier cu mici dimensiuni, care se va stoca pe terminalul unui utilizator(laptop, computer, telefon mobil etc.) prin intermediul caruia se accesează internetul și se facilitează accesul la anumite pagini web. Aceste cookie-uri sunt instalate prin solicitarea emisă de un server web către un browser (Internet Explorer, Opera, Google Chrome etc.). Un cookie este un program pasiv, însemnând ca nu conține programe software, viruși sau spyware și nu vor accesa informații de pe hard driverul utilizatorilor pe a căror echipament a fost instalat. Un cookie poate fi accesat din nou numai de către webserverul care l-a trimis atunci când un utilizator se reîntoarce pe wesite-ul din serverul respectiv.

Ce cookie-uri utilizeaza sigplast.ro?

Singurele cookie-uri pe care le folosim sunt cele implicite, necesare pentru a utiliza câteva funcții importante pe website-ul nostru, precum conectarea.

Cum se pot opri cookie-urile?

În general, dezactivarea cookie-urilor poate face ca anumite pagini de internet să fie dificil de utilizat, impunând anumite limitări utilizatorilor acestora. Pentru dezactivarea cookie-urilor puteți schimmba setările specifice din browserul pe care îl utilizați.
